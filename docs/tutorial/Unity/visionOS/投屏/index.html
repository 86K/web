投屏

支持将Apple Vision Pro上的画面镜像到iPhone、iPad、Mac、Apple TV或兼容“隔空播放”的智能电视机

官方地址

Apple Vision Pro投屏Macbook​

  1. apple vision pro与macbook在同一局域网，mac不能翻墙！！！

  2. 设置“隔空投送与接力”

  3. 在vision pro中，向上看，可较快的从上往下看，直至出现如下图，手指捏合，弹出步骤4的界面

    注：visionOS2.0之后，不需要向上看了，只需要翻转手心并捏合即可呼出控制中心。

  4. 点击图示的“控制中心”按钮

  5. 点击图示的“镜像”按钮

  6. 选择需要镜像的设备，在visionPro中输入设备密码

  7. 停止镜像
